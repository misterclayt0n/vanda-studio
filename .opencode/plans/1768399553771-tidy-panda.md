# Vanda Studio - Color and Typography Redesign Plan

## Overview
Reimagine the app's visual identity with warm, cream-toned backgrounds (inspired by Heidi/Convex), DM Serif Display headings (inspired by Svelte), while keeping pink as the signature brand color.

## Changes Summary

### 1. Add DM Serif Display Font
**File:** `src/app.html`

Add Google Fonts import for DM Serif Display:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=DM+Serif+Display:ital@0;1&family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### 2. Update CSS Variables
**File:** `src/app.css`

#### Font Variables (in `:root`)
```css
--font-sans: "Inter", ui-sans-serif, system-ui, sans-serif;
--font-serif: "DM Serif Display", Georgia, serif;  /* NEW */
--font-geist-mono: "Geist Mono", ui-monospace, monospace;
```

### 3. Add Tailwind Theme Token for Serif Font
**File:** `src/app.css` (in `@theme inline`)

```css
--font-serif: var(--font-serif);
```

### 4. Set Default to Light Mode
**File:** `src/routes/+layout.svelte`

Update ModeWatcher to default to light:
```svelte
<ModeWatcher defaultMode="light" />
```

### 5. Apply Serif Font to Headings
**File:** `src/app.css` (in `@layer base`)

Add serif font to h1, h2, h3 elements:
```css
@layer base {
  * {
    @apply border-border outline-ring/50;
  }
  body {
    @apply bg-background text-foreground font-sans antialiased;
  }
  h1, h2, h3 {
    font-family: var(--font-serif);
  }
}
```

**Note:** Keep `--radius: 0` for sharp corners as preferred.

## Color Palette Reference

### Light Mode Palette
- **Background:** Warm cream (#FDF8F0) - inspired by Heidi/Convex
- **Cards/Surfaces:** Slightly lighter cream (#FEFBF5)
- **Muted areas:** Darker cream (#F5EFE5)
- **Primary:** Pink (keeping existing oklch(0.59 0.22 1))
- **Text:** Near black (keeping existing)

### Dark Mode Palette
- **Background:** Deep charcoal with blue undertone (#1a1a2e style)
- **Cards/Surfaces:** Elevated charcoal
- **Primary:** Pink (keeping existing)
- **Text:** Off-white (keeping existing)
